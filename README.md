# Digital-LIteracy
Project file for digital literacy

Project Overview This repository serves as a comprehensive portfolio demonstrating my foundational skills in digital literacy, professional communication, and technical problem-solving. Through a series of five hands-on tasks, this project bridges the gap between academic learning and industry-ready digital habits, covering everything from online safety to algorithmic coding.

Task Breakdown Task 1: Digital Literacy Awareness Infographic Designed an educational infographic titled "Navigating the Digital World." This visual guide synthesizes core digital literacy concepts, curates essential productivity and research tools for students (like Notion, Google Workspace, and Mendeley), and outlines foundational safe internet practices.

Task 2: Student Digital Portfolio Setup Established a strong professional footprint across key industry platforms. This included configuring a GitHub profile (complete with a developer README), optimizing a LinkedIn profile for networking and internship scouting, and setting up a Kaggle account for applied data science practice.

Task 3: Coding & Collaboration Platforms (HackerRank) Practiced algorithmic problem-solving and mathematical logic by completing the "Modified Fibonacci" challenge on HackerRank. Developed an efficient Python script to compute rapidly growing sequences while managing variable states and preventing integer overflow errors.

Task 4: Professional Email & Etiquette Guide Developed a framework for professional digital communication. This task included drafting formal email templates for assignment extensions and internship inquiries, as well as analyzing a case study on the consequences of using informal channels (like WhatsApp) for critical academic project management.

Task 5: Cybercrime Awareness & Prevention Investigated the mechanics of modern cyber threats, specifically focusing on UPI payment fraud in online student marketplaces. Created an actionable "Stay Safe Online" checklist tailored for college students in India, covering 2FA implementation, QR code scams, and official reporting channels (Cybercrime.gov.in / 1930).
